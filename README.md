
Go to the project directory

```bash
  cd Tiktok-ViewsBOT
```

```bash
  npm install
```

Install dependencies modules

## Modules Termux

```bash
  pip install -r requirements.txt
```

```bash
  pip3 install -r requirements.txt
```

Start the server

```bash

```
